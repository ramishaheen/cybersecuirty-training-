Module 04 · Reference

The SLAM method

Four seconds. Four letters. Saves millions.

S
Sender
Hover the name. Is the domain real? paypal-secure.com ≠ paypal.com
L
Links
Hover every link. Mismatched display text = phishing.
A
Attachments
Unexpected .exe .iso .zip .docm? Don't open.
M
Message
Urgency, threats, grammar errors = red flag.

The seven deadly tells

!
Urgency — "in the next 2 hours", "today only"
!
Authority — "the CEO", "the Governor"
!
Secrecy — "don't tell anyone"
!
Look-alike domains — extra letters, dashes
!
Generic greeting — "Dear customer"
!
Off-channel — work topic via WhatsApp
!
Wire transfer / gift cards — never legitimate
!
Trust your gut — if off, report it
