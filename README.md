# Bank Al Etihad — Cybersecurity Essentials Training

Interactive one-day cybersecurity training deck delivered by Etihad Insights (Dr. Rami).

## Deploy

### Option A — Drag & drop
1. Go to https://vercel.com/new
2. Drag this whole folder onto the page
3. Click Deploy

### Option B — Vercel CLI
```bash
npm i -g vercel
cd vercel_deploy
vercel
```

### Option C — From GitHub
Push this folder to a repo, then import at vercel.com/new.

## Custom domain (optional)
After deployment, in Vercel dashboard → Project → Settings → Domains, add a custom domain like `cyber.etihadinsights.com`.
